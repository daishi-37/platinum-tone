# 作りたいもの
有料会員サイト（オンラインサロン）

# いまやりたいこと
まずは要件定義をしたい。から不足情報があればディスカッションして、マークダウン式の要件定義書にしたい

# 技術スタック
- フロントエンド: Next.js 14 (App Router)
- スタイリング: Tailwind CSS
- 認証 : Laravel Sanctumを利用した「セッション（Cookie）ベース認証」
- DB : MySql
- 仮想環境 : docker
- クレジット決済 : Stripe API

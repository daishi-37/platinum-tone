# 掲示板 機能仕様書

> 最終更新: 2026-06-04（実装完了・右パネルを回答一覧フィードに変更）

---

## 1. 概要

会員が講師へ質問を投稿できるQ&A形式の掲示板。  
左パネルはLINEグループチャット風のタイムライン、右パネルは**回答がついた質問を時系列で並べた「回答一覧」フィード**。  
講師はタイムラインに呼びかけを投稿することも、左の質問を選んで回答することもできる。

吹き出しは生徒の質問＝左寄せ・白、講師の呼びかけ／回答＝右寄せ・プライマリ色で、チャットらしく**しっぽ付き**で表示する。

---

## 2. 投稿の種類

| 種類 | 投稿者 | 表示場所 | 説明 |
|------|--------|---------|------|
| **質問** | 会員（生徒） | 左パネル・左寄せ | 講師への質問。回答がつくと右パネルの回答一覧にも表示される |
| **呼びかけ** | 管理者（講師） | 左パネル・右寄せ | 特定の質問に紐づかない講師からのアナウンス |
| **回答** | 管理者（講師） | 右パネル（回答一覧）のみ | 特定の質問に対する講師の回答。タイムラインには流れない |

---

## 3. アクセス権限

| 操作 | 会員 | 管理者（講師） |
|------|------|--------------|
| タイムライン閲覧 | ○ | ○ |
| Q&Aスレッド閲覧（全質問） | ○ | ○ |
| 質問の投稿 | ○（月20件まで） | × |
| 呼びかけの投稿 | × | ○（制限なし） |
| 回答の投稿 | × | ○ |
| 自分の質問の削除 | ○（回答なしの場合のみ） | ○（すべて） |
| 呼びかけの削除 | × | ○ |
| 回答の削除 | × | ○ |

**非会員・未ログイン**: 掲示板ページへのアクセス不可（ログインページにリダイレクト）

---

## 4. 投稿ルール

### 4-1. 質問（会員）

- 月20件まで投稿できる
- 制限は毎月1日 0:00（JST）にリセット
- 入会月・解約月も同様に20件まで投稿可能
- 投稿後の編集は不可
- 回答がついていない自分の質問は削除できる（削除後も月の投稿カウントは戻らない）
- 回答がついた後は削除不可（削除ボタン自体が非表示になる）

**バリデーション**:
- 必須入力
- 最大500文字
- 空白のみは不可

**投稿例**:
```
発声練習をするとき、どのくらいの時間を毎日かければ効果が出ますか？
最近始めたばかりで、1日15分程度しかできていないのですが、
もっと増やした方がよいでしょうか。
```

---

### 4-2. 呼びかけ（講師）

- 投稿件数に制限なし
- 投稿後の編集は不可
- 削除は管理者のみ可能

**バリデーション**:
- 必須入力
- 最大500文字

**投稿例**:
```
皆さん、今週末に発声ワークショップを開催します！
詳細はDMでお送りしますので、興味のある方はお知らせください。
```

---

### 4-3. 回答（講師）

- 管理者のみ投稿可能
- 1つの質問に複数の講師が個別に回答できる（上限なし）
- 回答はタイムラインに流れない（右パネルのQ&Aスレッドにのみ表示）
- 投稿後の編集は不可
- 削除は管理者のみ可能

**バリデーション**:
- 必須入力
- 最大2000文字

**回答が複数ついた場合の右パネル表示例**:
```
【田中さんの質問】
「発声練習はどのくらいやれば効果が出ますか？」

【仙台エリの回答】
毎日15分でも継続することが大切です。量より質を意識して...

【優希比呂の回答】
仙台エリさんの意見に加えて、朝起きてすぐにやるのがおすすめです...
```

---

### 4-4. 未回答の扱い

- 未回答の質問に特別なステータス管理は行わない
- 講師は全体を見て気になった質問に任意で回答する

## 5. 画面仕様

### 5-1. デスクトップ（左右2ペイン）

左パネル＝タイムライン（質問＋呼びかけ）、右パネル＝**回答一覧フィード**（回答がついた質問を時系列で全件表示）。  
右パネルは質問選択に依存せず常に全件を表示する。管理者が左の質問をクリックすると、右パネル下部にその質問への回答フォームのみが追加表示される。

```
┌──────────────────────────┬──────────────────────────────────┐
│  掲示板（タイムライン）   │  回答一覧（時系列・全件）          │
│  ──────────────────────  │  ──────────────────────────────  │
│ ◀ 田中さん               │  田中さん              2026/6/4  │
│  ┌────────────────────┐ │ ◀┌────────────────────────────┐ │
│  │「発声の練習は…」   │ │  │「発声を練習する際に…」     │ │
│  │ 6月4日 ✅ 回答あり │ │  └────────────────────────────┘ │
│  └────────────────────┘ │            仙台エリ ┌──────────┐▶│
│            仙台エリ ──┐ │                     │毎日15分…│ │
│      ┌────────────────┐▶│                     └──────────┘ │
│      │「皆さん今週末…│ │            優希比呂 ┌──────────┐▶│
│      └────────────────┘ │                     │朝起きて…│ │
│ ◀ 鈴木さん（クリックで  │                     └──────────┘ │
│  ┌────────────────────┐ │  鈴木さん              2026/6/3  │
│  │  回答対象に選択）  │ │ ◀┌────────────────────────────┐ │
│  └────────────────────┘ │  │「オーディション前日…」     │ │
│  ──────────────────────  │  └────────────────────────────┘ │
│  今月の残り: 17件        │            仙台エリ ┌──────────┐▶│
│  ┌────────────────────┐ │                     │まず喉を…│ │
│  │ 質問を入力…        │ │                     └──────────┘ │
│  └────────────────────┘ │  ── 〇〇さんの質問に回答（管理者）│
│  [投稿する]              │  ┌────────────────────┐ [回答する]│
└──────────────────────────┴──────────────────────────────────┘
   （◀▶ はチャット吹き出しのしっぽ）
```

### 5-2. モバイル（上部タブで切替）

画面上部の「タイムライン」「回答一覧」タブで2ペインを切り替える。  
管理者が質問をタップすると自動で「回答一覧」タブに切り替わり、下部の回答フォームから回答できる。

**タイムラインタブ**:
```
┌──────────────────────┐
│[タイムライン][回答一覧]│ ← タブ
│  ─────────────────   │
│ ◀ 田中さん           │
│  ┌──────────────┐   │
│  │「発声の練習は│   │
│  │ …」6/4 ✅回答│   │
│  └──────────────┘   │
│         仙台エリ ─┐  │  ← 呼びかけ（右寄せ・しっぽ右▶）
│      ┌───────────┐▶ │
│      │「皆さん…│   │
│      └───────────┘  │
│  今月の残り: 17件    │
│  ┌──────────────┐   │
│  │ 質問を入力…  │   │
│  └──────────────┘   │
│  [投稿する]          │
└──────────────────────┘
```

**回答一覧タブ**:
```
┌──────────────────────┐
│[タイムライン][回答一覧]│ ← タブ
│  ─────────────────   │
│ 田中さん        6/4  │
│ ◀┌──────────────┐   │
│  │「発声を練習…│   │
│  └──────────────┘   │
│       仙台エリ ─┐    │
│      ┌───────────┐▶ │
│      │毎日15分… │   │
│      └───────────┘  │
│       優希比呂 ─┐    │
│      ┌───────────┐▶ │
│      │朝起きて… │   │
│      └───────────┘  │
│ ── 回答フォーム（管理者）│
└──────────────────────┘
```

### 5-3. 吹き出しスタイル

| 投稿者 | 配置 | 背景色 | しっぽ | タップ時の動作 |
|--------|------|--------|--------|--------------|
| 生徒の質問 | 左寄せ | 白・グレーボーダー | 左向き（◀） | 管理者：回答対象に選択。会員：特になし |
| 講師の呼びかけ | 右寄せ | プライマリカラー（`#97d3c3`） | 右向き（▶） | 何も起きない |
| 講師の回答 | 右寄せ（回答一覧内） | プライマリカラー（`#97d3c3`） | 右向き（▶） | — |

吹き出しのしっぽは CSS（`frontend/app/globals.css` の `.chat-left` / `.chat-right`）で実装。白い質問バブルは枠線も再現するため三角形を2枚重ねている。

### 5-4. 質問バブルのバッジ

| 状態 | バッジ表示 |
|------|-----------|
| 回答あり | ✅ 回答あり（グリーン） |
| 回答なし | 表示なし |

---

### 5-5. 投稿フォームの出し分け

| ユーザー | 表示するフォーム |
|---------|----------------|
| 会員（生徒） | 質問投稿フォーム（左パネル下部）・残り件数表示 |
| 管理者（講師） | 呼びかけ投稿フォーム（左パネル下部）・残り件数表示なし |

- 管理者が左タイムラインの質問バブルをクリックした場合のみ、右パネル（回答一覧）下部にその質問への回答フォームが追加表示される
- 呼びかけバブルはクリックしても回答対象にならない
- 右パネルの回答一覧は質問の選択に関係なく常に全件を表示する

---

### 5-6. 削除ボタンの表示条件

**タイムライン上（左パネル）**:

| 状態 | 自分の質問 | 他人の質問 | 講師の呼びかけ |
|------|-----------|-----------|--------------|
| 会員ログイン・回答なし | 削除ボタンあり | 表示なし | 表示なし |
| 会員ログイン・回答あり | 表示なし | 表示なし | 表示なし |
| 管理者ログイン | 削除ボタンあり | 削除ボタンあり | 削除ボタンあり |

**削除確認ダイアログ例**:
```
質問を削除しますか？
「発声練習をするとき、どのくらいの...」

※ 削除すると元に戻せません。今月の投稿数にはカウントされたままです。

[キャンセル]  [削除する]
```

## 6. 通知仕様

**トリガー**: 会員が質問を投稿したとき（呼びかけ・回答では通知しない）

**通知先**: Slack チャンネルのメールアドレス（環境変数 `BOARD_SLACK_EMAIL`）

**メール件名**:
```
【tone 掲示板】新しい質問が届きました
```

**メール本文**:
```
tone 掲示板に新しい質問が投稿されました。

━━━━━━━━━━━━━━━━━━
投稿者: 田中さん
投稿日時: 2026年6月4日 14:32

質問内容:
発声練習をするとき、どのくらいの時間を毎日かければ
効果が出ますか？
━━━━━━━━━━━━━━━━━━

掲示板で確認・回答する:
https://tone-ac.com/board
```

## 7. エラー・制約一覧

| ケース | 表示メッセージ |
|--------|--------------|
| 質問の月上限到達 | 「今月の投稿上限（20件）に達しました。次回リセットは◯月1日です。」 |
| 本文が空 | 「内容を入力してください。」 |
| 500文字超過 | 「500文字以内で入力してください。」 |
| 回答済み質問を削除しようとした | （ボタン非表示のため到達しない） |
| 他人の質問を会員が削除しようとした | 403エラー（APIレベルで拒否） |
| 非会員がアクセス | ログインページへリダイレクト |

**作成日**: 2026-02-20

## 1. プロジェクト概要

### 1.1 サービス概要
選択理論心理学を用いた声優育成オンラインアカデミー「**tone**」。

グリーンノート所属の現役声優（仙台エリ・優希比呂）が直接指導。全国どこからでも声優業界のマインド・知識・スキルを学べる有料会員制 WEBアプリ。公開コンテンツと会員限定コンテンツを明確に分離し、月額課金によるサブスクリプションモデルで運営する。

### 1.2 主要ターゲット
- 声優を目指している人（全国）
- 声優業界の情報を継続的に得たい人

## 4. 会員プラン

### 4.0 プラン構成

| プラン | 種別 | 料金 | 内容 |
|---|---|---|---|
| **オンラインアカデミー** | 基本プラン（月額） | 9,200円（税込） | 全デジタルコンテンツへのアクセス |
| **グループレッスン** | オプション | 未定 | 別途申込。複数人でのオンラインレッスン |
| **個人レッスン** | オプション | 未定 | 別途申込。1対1のオンラインレッスン |

> オプション（グループ・個人レッスン）は基本プランへの加算または単体申込か、別途検討が必要。

### 4.1 基本プラン詳細

| 項目 | 内容 |
|---|---|
| 月額料金 | **9,200円（税込）** |
| 無料トライアル | 7 日間（Stripe の Trial Period を使用） |
| 決済サイクル | 月次（毎月自動更新） |
| 支払い方法 | クレジットカード（Stripe） |
| 解約 | ユーザーが任意のタイミングで解約可能。解約後は期間終了まで有料会員として継続 |

### 4.2 月額料金の算出根拠

| 項目 | 値 |
|---|---|
| 希望受取額（税込） | 8,800円 |
| Stripe 手数料率 | 3.6%（国内カード標準レート） |
| 計算式 | 8,800 ÷ (1 − 0.036) ≈ 9,129.7円 |
| 切り上げ（下二桁） | 9,200円 |
| **顧客請求額** | **9,200円（税込）** |
| 実際の受取額 | 9,200 × (1 − 0.036) ≈ 8,869円 |

> Stripe の手数料率はプランや契約によって変動する場合があります。本番導入前に Stripe ダッシュボードで自社の適用レートを確認してください。

## 5. コンテンツ仕様

### 5.1 公開コンテンツ（未ログインユーザーも閲覧可能）

| コンテンツ種別 | 詳細 |
|---|---|
| Podcast「声優登竜門バックステージ」 | mp3 をストリーミング再生（ダウンロード不可）。カスタムプレイヤーで実装 |
| Blog - What's 声優業界（公開記事） | 管理者が記事単位で「公開 / 会員限定」を選択できる |

### 5.2 会員限定コンテンツ（有料会員 / トライアル会員のみ）

| コンテンツ種別 | 詳細 |
|---|---|

| Podcast「声優登竜門バックステージ」深掘り版 | mp3 または Vimeo で提供。サムネイル付き。ダウンロード不可 |
| 動画 - 声優になるためのレッスン | 現場で活躍する声優も実践する基礎学習。Vimeo 限定公開 URL を埋め込み |
| 動画 - 対談（優希比呂 x 仙台エリ + ゲスト） | Vimeo 限定公開 URL を埋め込み |
| 動画 - 声優のマインドセット | Vimeo 限定公開 URL を埋め込み |

### 5.3 メディア保護方針
- mp3 は `<audio>` タグで再生のみ。直接 URL へのアクセスを Signed URL または Backend 経由のプロキシで防ぐ
- Vimeo は「ドメイン制限（埋め込み許可ドメイン限定）」設定を使用。ダウンロードボタン非表示
- 画像・サムネイルは CDN 経由で配信

## 8. デザイン方針

**参考サイト**: [TCD098 デモサイト](https://demo.tcd-theme.com/tcd098/) / [uki by non Editions](https://ukibynoneditions.com/)（色味の参考）

### 8.1 カラーパレット

| 役割 | カラー | 用途 |
|---|---|---|
| サイドバー背景 | `#2a3f38`（ソフトダークグリーン） | 左固定サイドバー |
| プライマリ | `#2a3f38`（ソフトダークグリーン） | ボタン・見出しアクセント |
| プライマリホバー | `#3d5c52` | ホバー・アクティブ状態 |
| ページ背景 | `#fafaf8`（温かみのあるオフホワイト） | 全体背景。紙のような柔らかさ |
| セクション背景 | `#f5f2ed`（ライトウォームベージュ） | セクション区切り |
| カード背景 | `#ffffff` | 記事・動画カード |
| 本文テキスト | `#2d2926`（ウォームダークブラウン） | 本文。重たくない温かみのある黒 |
| サブテキスト | `#928d87`（ウォームグレー） | 日付・カテゴリ・補足情報 |
| アクセント | `#7aaa94`（ミューテッドセージグリーン） | タグ・バッジ・ハイライト |

> **方針**: [uki by non Editions](https://ukibynoneditions.com/) のような「ふわっと軽い・温かみのある白」テイスト。冷たいティール系から暖色系オフホワイトに変更。サイドバーはソフトダークグリーンで落ち着いた重厚感を残す。

### 8.2 タイポグラフィ

| 項目 | 内容 |
|---|---|
| 本文フォント | Noto Sans JP（日本語）, Inter（英字） |
| 見出しフォント | Noto Sans JP Bold（日本語）/ Noto Serif JP（装飾見出し） |
| 本文サイズ | 15–16px（読みやすさ重視） |
| 見出しサイズ | レスポンシブ対応。PC: 36px（大見出し）、SP: 22px |
| 行間 | `line-height: 1.9`（日本語の可読性を確保） |

#### レイアウト仕様

| 項目 | 仕様 |
|---|---|
| 基本構造 | サイドバーなし。全ページ **フルワイド単カラム**ベース |
| コンテンツ幅 | 最大幅を `max-width` で制限（1200px 前後）し中央寄せ |

| アニメーション | スクロールトリガーで各要素をフェードイン |
| ヘッダー | 固定ヘッダー（スクロールしても追従）。PC/SP でロゴ切替 |

### 8.4 UI スタイル

| 要素 | 方針 |
|---|---|
| ボタン | 角丸（border-radius: 70px）。背景は `#003040`、テキストは白。ホバーで `#1a5068` に変化 |
| カード | 白背景 + 薄いシャドウ（`box-shadow: 0 2px 12px rgba(0,48,64,0.08)`）。角丸 8px |
| レイアウト | グリッド + Flexbox 基盤。ブレークポイントは 1200px / 768px |
| スライダー | トップページのフィーチャーコンテンツはカルーセル形式 |
| タブ | コンテンツセクションの切り替えにタブ UI を使用 |
| ディバイダー | セクション間は `#eaf0f2` の帯で自然に区切る |

### 8.4 全体トーン

- **軽やかな洗練感**: 背景・テキストを明るめに保ちつつ、ネイビーグリーンをポイントで効かせる
- **読みやすさ優先**: 純黒・純白を避け、チャコール × セージホワイトで目に優しい配色
- **コンテンツ重視**: 動画・音声・記事のサムネイルが引き立つよう、周囲は余白とライトカラーで整理

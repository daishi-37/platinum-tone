# プロジェクトメモリ

## 声優オンラインアカデミー プロジェクト

### デザイン参考サイト
URL: https://demo.tcd-theme.com/tcd098/
WordPress テーマ「TCD098 (SHIPS)」のデモサイト。レイアウト・カラー・UIの参考元。

### 参考レイアウト（トップページ構成）
1. フルワイドカルーセルスライダー（ヒーロー）
2. CTA セクション（キャッチ + 入会ボタン）
3. 価値訴求ブロック（3カラム 画像+テキスト）
4. サービス紹介テキスト
5. コンテンツカルーセル（タブ切替: 新着 / 会員限定 / 人気 / おすすめ）
6. 中間 CTA
7. 会員特典・実績（ドーナツグラフ等）
8. 会員の声（顔写真付きカルーセル）
9. バナーリンク 2本
10. フッター（5カラムナビ + SNS + コピーライト）

**構造的特徴**:
- サイドバーなし・フルワイド単カラム
- 固定ヘッダー
- Splide.js でカルーセル実装
- スクロールトリガーアニメーション
- ブレークポイント: 1221px / 850px
